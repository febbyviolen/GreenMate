# GreenMate

### 팀원 및 역할:
- 최민호: 팀장 및 계획
- 남권표: 백엔드
- 페비: 디자인 및 iOS 앱 개발
- 황유란: 안드로이드 앱 개발
- 이채린: 백엔드
- 정서현: 웹(랜딩 페이지)
